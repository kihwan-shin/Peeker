# Peeker

다른 프로세스의 윈도우를 실시간으로 캡쳐하여 항상 최상위(Always on Top)로 미리보기하는 Windows 유틸리티입니다.

## 요구 사항

- Windows 10/11 (x64)
- .NET 8.0 Runtime

## 설치

`PeekerSetup.exe`를 실행하여 설치합니다.

## 기본 사용법

### 프로세스 선택

1. Peeker를 실행합니다.
2. 화면을 **우클릭**하여 컨텍스트 메뉴를 엽니다.
3. **Select Window**를 클릭하고, 캡쳐할 프로세스의 윈도우를 선택합니다.

### 창 조작

| 동작 | 방법 |
|------|------|
| 이동 | 창을 드래그 |
| 크기 조절 | 창 가장자리를 드래그 |
| 컨텍스트 메뉴 | 우클릭 |

## 컨텍스트 메뉴

| 메뉴 | 설명 |
|------|------|
| **Select Window** | 캡쳐할 대상 윈도우 선택 |
| **Rename Profile...** | 현재 프로필 이름 변경 |
| **Set Region** | 캡쳐 영역을 드래그하여 지정 |
| **Reset Region** | 캡쳐 영역을 전체로 초기화 |
| **Interval** | 캡쳐 주기 설정 (50ms ~ 30초) |
| **Fit To Page** | 현재 창 크기에 맞게 이미지 비율 조정 (이미지 크기 변경 없음) |
| **Original Size** | 캡쳐 이미지 원본 크기(1:1)로 창 조정 |
| **Background Color...** | 배경색 변경 (색상 선택 다이얼로그) |
| **Show in Taskbar** | 작업 표시줄 표시 여부 토글 |
| **Run at Startup** | Windows 시작 시 자동 실행 등록/해제 |
| **About** | 버전 정보 |
| **Exit** | 종료 |

## 점프리스트

작업 표시줄의 Peeker 아이콘을 우클릭하면 최근 사용한 프로세스 목록이 표시됩니다. 클릭하면 해당 프로세스를 바로 캡쳐하는 새 Peeker 인스턴스가 실행됩니다.

## 커맨드라인

```
Peeker.exe <프로세스이름> [프로필이름]
```

- `프로세스이름`: 캡쳐할 프로세스 이름 (예: `notepad`)
- `프로필이름` (선택): 설정을 저장할 프로필 이름. 생략 시 프로세스 이름과 동일

## 시스템 트레이

Peeker는 시스템 트레이에 아이콘이 표시됩니다. 트레이 아이콘을 더블클릭하면 창이 복원됩니다.

## 설정 저장

프로필별로 다음 설정이 자동 저장됩니다:

- 창 위치 및 크기
- 캡쳐 영역 (Region)
- 캡쳐 주기
- 배경색
- Fit To Page 상태
- 작업 표시줄 표시 여부

설정 파일 위치: `%LocalAppData%\Peeker\<프로필이름>.json`
